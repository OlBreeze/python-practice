Задание: Разработка приложения StringProcessor

Разработайте консольное приложение StringProcessor, которое будет обрабатывать строки различными способами. Приложение должно иметь следующую функциональность:

1. Метод взаимодействия с пользователем через консоль, использующий класс Scanner для ввода данных. Метод должен запрашивать у пользователя строку и операцию для обработки этой строки.

2. Метод для парсинга вводимых данных, который будет разделять введенную пользователем информацию на строку и операцию.

3. Метод обработки строки в зависимости от выбранной операции. Реализуйте следующие операции:

4. Если пользователь введет неправильную операцию, попросите его ввести корректную операцию и повторите процедуру ввода данных с консоли.

5. Если пользователь введет пустую строку, попросите его ввести непустую строку и повторите процедуру ввода данных с консоли.

6. Реализуйте обработку ошибок с использованием исключений (Exceptions).

7. Добавьте возможность выхода из приложения при вводе специальной команды (например, "exit").
   - REVERSE: переворачивает строку
   - UPPERCASE: преобразует строку в верхний регистр
   - LOWERCASE: преобразует строку в нижний регистр
   - COUNT: подсчитывает количество символов в строке
   - TRIM: удаляет начальные и конечные пробелы
